# morris.js

Example usage [morris.js][0].

[Live Demo][1]

*Django 1.7 e Python 3.4*

	$ git clone https://github.com/rg3915/morris.git
	$ virtualenv -p python3 morris
	$ cd morris
	$ source bin/activate
	$ make setup
